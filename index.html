Student Smart Portal
The project titled as Student Smart Portal is a web based application. The growing significance of websites for various organization is well known. In recognizing that, we decided to develop a website for the students of our department i.e, Computer Science Engineering & Application (CSEA) which comprises of all the required information about the department. This application provides a virtual tour of the department. The main purpose of our project is to design, publish and maintain a website for our department which contain all the information regarding the department like courses, academic syllabus, lecture notes, internships & placements etc. This generic application designed for assisting the students of the department regarding information on courses, subjects, time tables, semester and results. The functionality requirements of the website are it provide all the information about the department, all necessary information is rendered with proper images and flash slides for better navigation. Moreover it provides all the latest updates and announcement about the recent events, notices and placement drive. Basically this website will be the key for the students. The software requirement for this project are Browser like Google Chrome, Database like SQLite, and Operating system like Windows. The technology used are HTML, CSS, and JavaScript for the frontend part and Django framework using Python programming language for the backend part. From this project, we hope that the students will get a good source of information about their department.
